Description

The basics of computer programming concepts using the C programming language. Emphasis on problem solving and algorithm implementation using a universal subset of the C programming language. Lecture and laboratory.

Purpose of C programming

This course is designed for engineering students who are comfortable working by computer. This course doesn't require any former knowledge about computer, But it requires enthusiasm and commitment to learn.



This course focuses on problem solving, design algorithms and finally implementation by C programing language. By end of this class, you will be familiar with concepts:

  • Varaibles, Data Type, Arithmetic and Logic Operators
  • Conditions, Control Structures
  • Functions
  • Array
  • File (Read/Write)

These concepts are general and can be used in other high level programming languages such as C++, C#, Java,... you can read more about C programming syllabus here
